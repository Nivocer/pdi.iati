.. _glossary:

Glossary
========

.. glossary::
   :sorted:
      
   UML
      UML stands for `Unified Modeling Language`_, used to specify behaviour 
      and design of software. We use PlantUML_, which is available as 
      extension_ for Sphinx_, to produce the UML diagrams in this documentation.
      
      .. _`Unified Modeling Language`: http://en.wikipedia.org/wiki/Unified_Modeling_Language
      .. _PlantUML: http://plantuml.sourceforge.net
      .. _extension: https://pypi.python.org/pypi/sphinxcontrib-plantuml
      .. _Sphinx: http://sphinx-doc.org

   IATI Standard
      An open data standard for the exchange of information on activities
      in international development cooperation. See the `IATI Standard`_
      website for more information.
      
      .. _IATI Standard: http://www.iatistandard.org
